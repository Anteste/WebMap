* Perform A Network Scan using Nmap
* Perform A Vulnerabity Scan Using Nikto
* Perform A Directory Busting using Using Dirsearch
* Generate Reports
* Automate Web Penetration Using WebMap
* Tested on Linux and Unix with Python 3
